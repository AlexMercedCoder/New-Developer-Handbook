## Graph Database

A Graph Database is a database that tracks all data as nodes and edges.

For example:

node 1
```
| Alex |
- name: "alex merced"
```

node 2
```
| Tony |
- name: "Tony Merced"
```

edge 1 & 2
```
| Alex | ----brotherOf---> | Tony |
| Tony | ----brotherOf---> | Alex |
```

- ArangoDB
- Fauna
- dGraph
- GunDB
